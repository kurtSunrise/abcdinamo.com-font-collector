Quick Configuration

Edit these values in the script's CONFIG object before running:

const CONFIG = {
    autoDownload: true,           // true = download files, false = get base64
    batchSize: 10,                 // Number of fonts to download at once (5-20)
    batchDelay: 100,               // Delay between batches in ms
    filterFamilies: ['Diatype'],   // Only download specific families (empty = all)
    excludeFamilies: [],           // Exclude these families
    verbose: true,                 // Show detailed progress
    showSummary: true              // Show summary at end
};

Common Examples

Download Only Diatype Fonts

filterFamilies: ['Diatype']

Exclude Language Variants

excludeFamilies: ['Thai', 'Hebrew', 'Arabic', 'Georgian', 'Armenian', 'Devanagari', 'Greek']

Fast Download

batchSize: 20, batchDelay: 50

Slow & Stable

batchSize: 5, batchDelay: 500

Get Base64 Data (No Downloads)

autoDownload: false

Troubleshooting

Fonts not downloading?

Make sure you're on https://abcdinamo.com/typefaces/diatype

All downloads failing (403 Forbidden)?

Some fonts aren't publicly accessible. Only ~50-60 fonts will download successfully.

Browser crashing?

Reduce batchSize to 3-5 and increase batchDelay to 1000

"Failed to fetch" errors?

CDN may be rate-limiting. Increase batchDelay to 500-1000ms

Only got 51 fonts?

That's normal! Some fonts require visiting specific typeface pages.
